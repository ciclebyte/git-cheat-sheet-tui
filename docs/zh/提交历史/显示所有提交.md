# 显示所有提交

## 一、命令介绍

`git log --oneline` 命令用于显示当前分支的所有提交记录，每条记录仅显示提交的哈希值和提交信息。

## 二、命令使用

### 命令格式

```bash
git log --oneline
```

### 命令示例

```bash
git log --oneline
```

## 三、输出说明

以下是执行该命令后可能输出的提交记录及对应解释：

```bash
# 提交记录的哈希值和提交信息
1234567 Update README.md
abcdefg Fix bug in main function
```

## 四、注意事项

### 提交记录的显示顺序

默认情况下，`git log` 命令会按提交时间的倒序显示提交记录，即最新的提交记录会显示在最前面。

### 限制提交记录的显示数量

可以通过增加 `-n` 参数来限制显示的提交记录数量，例如 `git log --oneline -n 5` 将仅显示最新的5条提交记录。
